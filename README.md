# 台灣捷運招考題庫

北捷・桃捷・中捷・高捷 招考考古題搜尋與線上測驗練習網站（純靜態網頁，無需伺服器）。

## 功能

- **題庫搜尋**：2600+ 題官方歷屆試題（程式解析自官方 PDF），依公司、年度、科目、關鍵字搜尋，逐題附正確答案與來源
- **測驗專區**：隨機抽題模擬作答、交卷計分（100 分制）、逐題檢討、歷史成績（localStorage）
- **官方考古題**：整理近五年（民國 110～114 年）各公司招考概況與官方試題 PDF 下載入口

## 使用方式

直接用瀏覽器開啟 `index.html` 即可，不需安裝任何東西。

## 檔案結構

```
index.html   # 介面（單頁式應用）
style.css    # 樣式（深色捷運路線風）
app.js       # 搜尋、測驗、計分邏輯
questions.js # 完整題庫（2600+ 題，程式解析自官方 PDF）
data.js      # 公司／年度／官方連結資訊
tools/       # PDF 下載與解析腳本（parse_trtc/tmrt/tymetro、build_questions）
raw/         # 原始官方 PDF 與中繼 JSON（未上傳，見 .gitignore）
```

## 資料來源聲明

- 題目取自各捷運公司**公開試題**（台北捷運官方試題專區、台中捷運官方歷屆試題專區等）與網路公開題庫（阿摩線上測驗、鼎文題庫網等），每題均附來源連結
- 高雄捷運採隨缺隨招、不對外公布試題，故本站未收錄高捷真題（不編造假題），僅提供官方徵才連結與共同科目練習引導
- 本站為**非官方**練習平台，題目與答案以各公司官方公布為準
- 不蒐集任何個人資料；測驗成績只存在使用者自己的瀏覽器
